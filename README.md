# VryfiExchange Vercel App

This folder contains a minimal React/Vite version of the landing page, ready to push to GitHub and host on Vercel.

## Files

- `index.html`
- `src/App.jsx`
- `src/main.jsx`
- `src/styles.css`
- `public/founders/*`
- `package.json`
- `vite.config.js`

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Create a new GitHub repo.
2. Upload the contents of this folder to the repo root.
3. Import the repo into Vercel.
4. Framework preset: `Vite`
5. Build command: `npm run build`
6. Output directory: `dist`

That should be enough to get the Beehiiv embed working on a real hosted URL.
